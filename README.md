## Propósito

El presente proyecto tiene como finalidad el desarrollo de un portal web utilizando tecnologías HTML, CSS, JavaScript y el framework Bootstrap, aplicando los conceptos de diseño web, estructura de páginas y experiencia de usuario.

## Descripción del proyecto

Se desarrolló el sitio web Clínica Veterinaria Huellitas, orientado a brindar información clara y organizada sobre los servicios veterinarios ofrecidos en el municipio de Carepa.

El portal incluye las siguientes secciones:

- Inicio  
- Nosotros  
- Instalaciones  
- Servicios  
- Contacto  

Además, permite a propietarios de mascotas conocer los servicios, tarifas, información institucional y solicitar citas mediante un formulario.

## Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Bootstrap  

## Características del sitio

- Diseño responsive (adaptable a dispositivos móviles)  
- Navegación clara e intuitiva  
- Interfaz moderna utilizando Bootstrap  
- Secciones informativas organizadas  
- Formulario de contacto para solicitud de citas  
- Accesibilidad que permite la navegación mediante lectores de pantalla como TalkBack
